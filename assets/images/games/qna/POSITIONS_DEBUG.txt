=== GUIDE DE POSITIONNEMENT DES ASSETS QNA ===

Mode Debug : Appuyez sur Alt+I dans le jeu pour activer/désactiver le mode debug

Le mode debug affiche en temps réel les positions X,Y de chaque asset :
- Background
- Door
- Wheel Outer
- Wheel Center
- Tube Front

ORDRE D'AFFICHAGE (arrière vers avant) :
1. Background (images/games/qna/background.png)
2. Door (images/games/qna/door.png)
3. Wheel Outer (images/games/qna/wheel-outer.png)
4. Wheel Center (images/games/qna/wheel-center.png)
5. Tube Front (images/games/qna/tube-front.png)

POSITIONS INITIALES PAR DÉFAUT :
Ces valeurs sont définies dans QuestionAnswerGameScreen.updateAnimationPositions()

Background: (0, 0)
Door: (centré - ajusté selon viewport)
Wheel Outer: (centré - ajusté selon viewport)
Wheel Center: (centré - ajusté selon viewport)
Tube Front: (centré - ajusté selon viewport)

COMMENT AJUSTER LES POSITIONS :
1. Lancez le jeu QNA
2. Appuyez sur Alt+I pour activer le mode debug
3. Notez les positions X,Y affichées à l'écran
4. Reportez les valeurs dans le code :
   Fichier : core/src/main/java/com/widedot/calendar/screens/QuestionAnswerGameScreen.java
   Méthode : updateAnimationPositions()
   Lignes : ~956-977

EXEMPLE DE CODE À MODIFIER :
```java
float backgroundX = 0f;
float backgroundY = 0f;

float doorX = (viewportWidth - 400f) / 2;
float doorY = (viewportHeight - 600f) / 2;

float wheelOuterX = (viewportWidth - 300f) / 2;
float wheelOuterY = (viewportHeight - 300f) / 2;

float wheelCenterX = (viewportWidth - 150f) / 2;
float wheelCenterY = (viewportHeight - 150f) / 2;

float tubeFrontX = (viewportWidth - 250f) / 2;
float tubeFrontY = (viewportHeight - 400f) / 2;
```

Note : Les positions sont relatives au viewport du jeu (DisplayConfig.WORLD_WIDTH x viewport.getWorldHeight())
